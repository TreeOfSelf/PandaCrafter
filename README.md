# Panda Crafter

## Description

A drop-in server-side mod that adds a simpler to use auto-crafter.  
Simply craft by putting a dropper over a Crafter block.   
This creates an "Easy Crafter" that looks like a dropper.  
Put the recipe pattern in the dropper, place a storage block behind it with ingredients (like a chest), then power to produce recipe reuslt.   

Originally inspired by https://modrinth.com/mod/eac
